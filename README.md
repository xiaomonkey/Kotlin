# kotlin

###什么是kotlin?

    kotlin是一门可以运行在java虚拟机、android、浏览器上的静态语言它与java100%兼容，如果对java非常熟悉，那么就会发现Kotlin除了自己的标准库之外，大多使用的是经典的java集合框架
    
    
    
###Kotlin的发展史

- 2010 立项
- 2011年6月，对外公开
- 2012年开源
- 2013年8月支持androidstudio
- 2014年6月全新开源的web站点和域名Kotlinlang.org
- 2016年2月发布正式版1.0
- 2016年9月发布1.0.4，支持apt
